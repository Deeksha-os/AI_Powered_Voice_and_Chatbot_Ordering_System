# AI-Powered Voice and Chatbot Ordering System

## Project info

**URL**: https://lovable.dev/projects/25fb648a-6e83-42ea-a923-3666d79f300e
The AI-Powered Voice and Chatbot Ordering System is a web application that allows users to order groceries using voice commands or chatbot interactions.
The system connects customers with nearby local vendors and uses AI-based recommendations and demand forecasting to improve efficiency and user experience.

# Aim
To build an AI-driven voice and chatbot ordering platform that enables fast, accessible, and personalized hyperlocal grocery ordering.

# Features
Voice-based grocery ordering
Chatbot-based text ordering
AI-powered product recommendations
Vendor inventory & order management
Admin dashboard
Role-based authentication (Admin, Vendor, Customer)

# Tech Stack
Frontend: React, HTML, CSS, JavaScript, Tailwind CSS
Backend: Python, Flask
Database: SQLite
AI / ML: NLP, Speech Recognition
